## Features

- 🔗 **Link Extraction** - Automatically detect links from HTML or plain text
- ✅ **Select/Deselect** - Choose which links to open or download
- 📥 **Download** - Download selected files directly
- 🚀 **Open Multiple** - Open selected links in new tabs simultaneously
- 🔒 **Privacy First** - No server, no tracking, everything runs locally
- 📱 **Responsive** - Works on desktop and mobile devices
- 🌍 **i18n** - English and Korean support


### Supported Input Types

| Type | Description |
|------|-------------|
| HTML | `<a href="...">` and `<area href="...">` tags |
| Plain Text | URLs starting with `http://`, `https://` |


## Usage

1. **Paste Content** - Copy HTML or text containing links and paste into the text area
2. **Review Links** - Detected links appear as a checklist
3. **Select Links** - Check/uncheck the links you want
4. **Take Action**:
   - **Download** - Download selected files
   - **Open** - Open selected links in new tabs


## Limitations

- **CORS Restrictions** - External file downloads may be blocked by browser security policies
- **Popup Blockers** - Opening multiple links requires popup permission

## 기능

- 🔗 **링크 추출** - HTML이나 일반 텍스트에서 링크 자동 감지
- ✅ **선택/해제** - 열거나 다운로드할 링크 선택
- 📥 **다운로드** - 선택한 파일 직접 다운로드
- 🚀 **여러 개 열기** - 선택한 링크를 새 탭에서 동시에 열기
- 🔒 **개인정보 보호** - 서버 없음, 추적 없음, 모든 것이 로컬에서 실행
- 📱 **반응형** - 데스크톱과 모바일에서 작동
- 🌍 **다국어** - 한국어 및 영어 지원


### 지원 입력 유형

| 유형 | 설명 |
|------|------|
| HTML | `<a href="...">` 및 `<area href="...">` 태그 |
| 일반 텍스트 | `http://`, `https://`로 시작하는 URL |


## 사용법

1. **내용 붙여넣기** - 링크가 포함된 HTML이나 텍스트를 복사하여 텍스트 영역에 붙여넣기
2. **링크 확인** - 감지된 링크가 체크리스트로 표시됨
3. **링크 선택** - 원하는 링크 체크/해제
4. **작업 실행**:
   - **다운로드** - 선택한 파일 다운로드
   - **열기** - 선택한 링크를 새 탭에서 열기


## 제한사항

- **CORS 제한** - 외부 파일 다운로드는 브라우저 보안 정책에 의해 차단될 수 있습니다
- **팝업 차단** - 여러 링크 열기는 팝업 권한이 필요합니다
